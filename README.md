## Use

```bash
npm i --save-exact @codemachiner/m
```

## Install

```bash
git clone git@github.com:codemachiner/m.git && \
  cd m && \
  npm run setup
```

## Develop

Use `npm test` to run tests (any `*.test.js`) once or `npm run tdd` to watch `src` folder for changes and run test automatically.
